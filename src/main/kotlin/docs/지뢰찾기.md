# **🚀 1단계 - 지뢰 찾기(그리기)**

---

## 기능 요구사항

지뢰 찾기를 변형한 프로그램을 구현한다.

- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- **지뢰는 눈에 잘 띄는 것으로 표기한다.**
- **지뢰는 가급적 랜덤에 가깝게 배치한다.**

## 실행결과

```console
높이를 입력하세요.
10

너비를 입력하세요.
10

지뢰는 몇 개인가요?
10

지뢰찾기 게임 시작
C C C * C C C * C C
C C * C * C C C C C
C C C C C C C C C C
C C C C C C C C C C
* C C C C C C C C C
C C C C C C * C C C
C C * C C C * C C C
C C C C C C * C C *
C C C C C C C C C C
C C C C C C C C C C
```

---
### 기능 구현 목록
- 높이
  - [x] 0보다 큰 양의 정수여야 한다.
- 너비
  - [x] 0보다 큰 양의 정수여야 한다.
- 지뢰 개수
  - [x] 0보다 커야한다.
  - [x] 높이x너비 한 값보다 작아야한다.
- 보드
  - [x] 지뢰 개수만큼 랜덤의 지뢰 좌표를 생성한다.
- 셀 생성기 (지뢰, 빈칸)
  - [x] 숫자 리스트로 들어온 값으로 지뢰 또는 빈칸의 셀을 생성한다.
